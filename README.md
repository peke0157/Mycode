# 音響解析（環境音・音楽ジャンル判定）システム
収録された音声データから収録された音声が音楽なのか、環境音なのか分析するシステムです。また、音楽であればその音楽がどのジャンルに位置するかを判定します。音響解析システムは**Gradio**を使って使用します。

# ソースコード解説
始めにテスト・学習用の音声データは「GTZAN Dataset」、「ESC-50」、「FREE BGM DOVA-SYNDROME」を使用しています。
